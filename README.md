# HoverBird

A 2D Flappy Bird clone built with **Phaser 3** and JavaScript. This project was created as a learning exercise to understand game development concepts, arcade physics, and the Phaser framework.

<!--  ## 🕹️ Live Demo -->

## 🚀 Features
* Simple one-button input (Spacebar/Touch) to flap.
* Procedural obstacle generation with random heights.
* Arcade physics system handling gravity and collisions.
* Score tracking and game-over states.

## 🎓 Learning Objectives
This project was built while following the course: **Game Development in JS/TS - The Complete Guide (w/ Phaser 3)** by **Filip Jerga**. 

Key concepts practiced:
* Implementing Phaser 3 Scenes (Preload, Game, Game Over).
* Working with `Phaser.Physics.Arcade`.
* Managing game loops and state updates.
* Asset loading and sprite manipulation.

## 🛠️ Installation & Setup
To run this game locally:

1. Clone the repository:
   ```bash
   git clone https://github.com/MorphBot/HoverBird
   ```
2. Open the directory:
   ```bash
   cd HoverBird
   ```
3. Use Node.js using nvm and install packages
    ```bash
    nvm install
    npm install
    ```
4. Run a local web server (Phaser requires a server to load assets properly):
   ```bash
   npm run dev
   ```
5. Open your browser and navigate to `http://localhost:8080`
